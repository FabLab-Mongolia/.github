# GerLab  
## FabLab Mongolia · Making, Learning & Sharing

Welcome to **GerLab**, a FabLab and creative fabrication space in Mongolia.

GerLab brings together digital fabrication, engineering, science, design, education, and Mongolian maker culture. We believe in one simple idea:

> **Learning by doing.**

This GitHub organization collects GerLab’s open knowledge, machine guides, safety rules, tutorials, material notes, project records, and community work.

GerLab is not only a room with machines.  
It is a place where ideas become physical objects, mistakes become lessons, and people build together.
